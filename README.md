# HTTP server websocket support

Adds websocket support to httpserver.

## Usage

Add the maven dependency:

    <dependency>
        <groupId>codemonstur</groupId>
        <artifactId>httpserver-websocket</artifactId>
        <version>0.1.0</version>
    </dependency>
